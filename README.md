# Small Cat (scat)
A small `cat` program written in assembly for my amusement

### Build instructions
nasm -f bin scat.asm  
From this you get a 64bit linux binary

### F.A.Q
Q: Why is it called Small Cat?  
A: Because it's small. And because it's funny.
